FlexBox Properties

Align-Item

Align-Item property controls the alignment of flex items on the cross axis.

Here flex:row => the cross axis = vertical axis and for flex:column => the cross axis = horizontal axis.



Justify-Content

Justify-Content property controls the alignment of flex items on the main axis of the flex container.

Here flex:row => the main axis = horizontal axis and for flex:column => the main axis = vertical axis.



Align-Content

Align-Content property sets the distribution of space between and around content items when there is extra space in the cross axis, similar to how 'justify-content' aligns individual items within the main-axis.

Here flex:row => the cross axis = horizontal axis and for flex:column => the cross axis = vertical axis.
